GLOBALCRAFT

GlobalCraft is a Minecraft Earth SMP server where players can explore a 1:500 scale map of the real world. Build your nation, form alliances, engage in wars, and trade with others. How to join

Here you can see how you can join the Minecraft server.

1

Add the server

Go to Multiplayer in Minecraft, then Add Server
2

Enter server address

Enter globalcraft.on.lightwi.re in the server address field and click "Done".
3

You're done!

Story about Globalcraft

Globalcraft was founded on February 26, 2025 by visionary founders Muf4ld and Luka. Sponsored by LightwireHosting, the server was built to deliver a unique, community-driven Minecraft experience. From the beginning, Globalcraft has been designed to foster creativity, challenge competitive gameplay, and build strong alliances. Today, players from around the world join in to explore.

Globalcraft supports versions from 1.8 up to 1.21. Whether you’re playing on a slightly older version or the latest release, you can join in on the adventure.
